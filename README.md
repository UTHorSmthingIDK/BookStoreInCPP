# BookStore in C++

A simple console-based bookstore management system written in C++.

This project is made for practice and learning purposes and is still under development.

## Features
- Manage authors
- Manage books
- File persistence
- ANSI colors
- UTF-8 console support (Windows)

## Status
The Program still has bugs

## UTF-8 (Windows)
If you are using Windows, you may need to run the following command
before executing the program:

```bash
chcp 65001
```

## Build
```bash
g++ BookStore.cpp -o BookStore
```

## Run
```bash
./BookStore
```
